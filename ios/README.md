# Distill iOS (Deprecated)

> **This SwiftUI client is deprecated and unmaintained.** Distill moved to a web
> app to avoid the paid Apple Developer account that push notifications and
> Sign in with Apple require. See
> [`docs/adr/0003`](../docs/adr/0003-web-client-and-email-delivery.md).

The active client lives in [`../web`](../web). This tree is kept for reference
only — it still targets Sign in with Apple + FCM push, which the backend no
longer supports (push was replaced by email delivery).
